# E-Learning Platform - React Frontend

A comprehensive React-based frontend for an e-learning platform with support for students and instructors.

## Features

### For Students
- User registration and authentication
- Browse and enroll in courses
- Track learning progress
- Take quizzes and assignments
- Earn certificates upon course completion
- View dashboard with stats and progress

### For Instructors
- Create and manage courses
- Add lessons and quizzes
- Track student progress
- View detailed analytics
- Generate certificates for students

## API Integration

This frontend integrates with a Django REST API backend. The complete API documentation with all 40+ endpoints is provided in the original documentation.

### Key API Categories
1. **Authentication APIs** - User registration and login
2. **Course APIs** - Course management and enrollment
3. **Lesson APIs** - Lesson content management
4. **Quiz APIs** - Quiz creation and attempts
5. **Progress Tracking APIs** - Student progress monitoring
6. **Certificate APIs** - Certificate generation and verification
7. **Dashboard APIs** - Student and instructor dashboards
8. **Analytics APIs** - Course and student analytics
